  \begin{definition}[Kronecker delta function]
  \label{def:kronecker}
    \textit{Kronecker delta function} is a function of two variables, usually
    integers, which is 1 if they are equal, and 0 otherwise.
    Formally:

    \begin{equation}
      \delta_{ij} = \begin{cases}
        1 & \text{if } i = j \\
        0 & \text{if } i \neq j
      \end{cases}
    \end{equation}
  \end{definition}
