% | caption, subcaption - Customization of captions in floating environments.
% |
% | The caption package provides many ways to customize the captions in floating environments like 
% | figure and table.
% | It also supports captioning of non-floating content through \captionof command.
% | The subcaption package allows for the use of sub-figures or sub-tables within a single figure or
% | table environment. 
% | It is a complementary extension to the caption package.
% |
% | No parameters are used with these package declarations.
\usepackage{caption, subcaption}
